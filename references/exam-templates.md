# 组卷模板库

## 使用说明

根据用户要求（年级/时长/难度/区域/题型偏好），自动匹配模板生成试卷。

所有生成的试卷必须包含：试题+答案+解析+考点标签。

## 组卷时自动执行

1. 确定年级 → 确定语法范围
2. 确定区域 → 确定难度系数
3. 确定时长 → 匹配模板
4. 确定目标 → 生成试题
5. 每道题标注：考点标签 + 难度 + 来源(真题/改编/原创) + 解析
