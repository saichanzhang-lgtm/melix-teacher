# 教材知识库

## 小学教材

### 北京版（北京教育出版社）— 昌平/顺义/通州主流

| 年级 | 核心语法 | 核心词汇量 | 核心句型示例 |
|------|----------|------------|-------------|
| 一上 | 无明确语法 | ~80词 | Hello! / Good morning! / What's your name? |
| 一下 | 名词单复数启蒙 | ~100词 | How many...? / I like... |
| 二上 | Be动词 (am/is/are) | ~120词 | This is... / I'm... / He's... |
| 二下 | 现在进行时启蒙 | ~130词 | What are you doing? / I'm reading. |
| 三上 | have/has 用法 | ~150词 | I have a... / Do you have...? |
| 三下 | 一般现在时入门 | ~160词 | I usually... / She always... |
| 四上 | 一般现在时第三人称单数 | ~180词 | He goes... / She doesn't... |
| 四下 | There be 句型 | ~190词 | There is/are... / Is there...? |
| 五上 | 一般过去时（规则动词） | ~200词 | I played... / Did you...? / I didn't... |
| 五下 | 一般过去时（不规则动词） | ~220词 | went, saw, ate, bought, took |
| 六上 | 一般将来时 (will / be going to) | ~250词 | I will... / Are you going to...? |
| 六下 | 四种时态综合复习+比较级 | ~280词 | 综合运用+衔接初一 |

### 人教PEP版 — 海淀/西城/朝阳部分小学

| 年级 | 核心语法 | 与北京版差异 |
|------|----------|-------------|
| 三起 | 较晚起步，三上才开始学英语 | 北京版一年级起步 |
| 五下 | 过去时稍晚引入 | 北京版五上已引入 |
| 六下 | 比较级+一般将来时 | 内容略少于北京版六年级 |

**关键差异**：PEP版三年级起点，词汇量和语法深度均略低于北京版同年级。海淀区部分学校会补充额外阅读材料。

### 外研版（一年级起点）— 部分重点小学

特点：一年级起步、阅读量大、词汇量要求高（比北京版多约15-20%）。部分国际学校和重点小学使用。

### 北师大版初中英语

特点：话题驱动型、重视听说、语法螺旋式上升。海淀部分学校和西城部分学校使用。
与PEP主要差异：每单元语法聚焦更集中，语法板块更明确。

### 外研版初中英语

特点：模块化设计、阅读量大、注重跨文化。部分区县使用。

## 高中教材

### 人教版高中英语（2019版）

- 必修1-3 + 选择性必修1-4
- 语法覆盖面与旧版相近
- 核心：非谓语动词（to do/doing/done）贯穿三册必修
- 高考语法填空考点全覆盖

### 北师大版高中英语

- 与PEP难度相当
- 编排更偏主题式
- 语法体系完整度略低于PEP

## 通用易错点速查

### 小学阶段 TOP10 易错

| # | 易错点 | 示例 |
|---|--------|------|
| 1 | 三单忘记加-s | ❌ He go → ✅ He goes |
| 2 | 过去式didn't后+过去式 | ❌ didn't went → ✅ didn't go |
| 3 | 名词复数不规则 | ❌ childs → ✅ children |
| 4 | there be与have混淆 | ❌ There have → ✅ There is/are |
| 5 | a/an选择 | ❌ a apple → ✅ an apple |
| 6 | 物主代词形容词性/名词性 | ❌ This is mine book → ✅ my book |
| 7 | 现在进行时缺be | ❌ I going → ✅ I'm going |
| 8 | 时间介词in/on/at | ❌ at Monday → ✅ on Monday |
| 9 | some/any混用 | ❌ I don't have some → ✅ any |
| 10 | 比较级忘记than | ❌ taller me → ✅ taller than me |

### 初中阶段 TOP10 易错

| # | 易错点 | 示例 |
|---|--------|------|
| 1 | 现在完成时与一般过去时混淆 | 有具体过去时间→必须用过去时 |
| 2 | 宾语从句语序 | ❌ I know where is he → ✅ where he is |
| 3 | if条件句时态 | ❌ If it will rain → ✅ If it rains |
| 4 | 被动语态缺be | ❌ The book wrote → ✅ was written |
| 5 | used to vs be used to | to do vs to doing |
| 6 | 比较级双重标记 | ❌ more taller → ✅ taller |
| 7 | 主将从现 | ❌ I'll call you when I will arrive → ✅ when I arrive |
| 8 | 非谓语动词 to do vs doing | ❌ enjoy to swim → ✅ enjoy swimming |
| 9 | 定语从句关系词 | ❌ who/that/which混淆 |
| 10 | 反意疑问句 | ❌ He is tall, isn't he? (助动词一致→需注意特殊) |

### 高中阶段 TOP10 易错

| # | 易错点 | 示例 |
|---|--------|------|
| 1 | 非谓语动词作状语/定语 | doing vs done vs to do 逻辑主语判断 |
| 2 | 虚拟语气时间倒退 | 与现在/过去/将来相反的时态倒退 |
| 3 | 倒装句部分倒装 | Only then did I realize... |
| 4 | 定语从句关系词省略 | 什么时候可以省略that/which/who |
| 5 | with复合结构 | with + 宾语 + doing/done/to do/adj |
| 6 | 情态动词+have done | must have done vs should have done vs could have done |
| 7 | 独立主格结构 | 名词/代词 + 分词/不定式/形容词 |
| 8 | as/which引导非限制性定语从句 | which可指代整句话，as有"正如"之意 |
| 9 | 强调句vs定语从句 | It is...that/who vs 关系词 |
| 10 | 名词性从句that不省略 | 主语从句中that不可省 |
