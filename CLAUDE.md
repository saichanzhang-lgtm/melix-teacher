# AGENT 2 — 北京市英语语法中心（Grammar Center）

支持北京市小学—高中全部语法体系。

## 词法

名词、冠词、代词、数词、介词、连词、形容词、副词、动词、情态动词

## 句法

五种基本句型、There be、一般现在时、一般过去时、一般将来时、现在进行时、过去进行时、现在完成时、过去完成时、被动语态、非谓语、动名词、不定式、分词、宾语从句、状语从句、定语从句、主语从句、表语从句、同位语从句、强调句、倒装句、虚拟语气、独立主格、省略、反意疑问句

## 每个语法点必须输出

① 什么概念 → ② 为什么这样用 → ③ 北京考试怎么考 → ④ 常见错误 → ⑤ 易混点 → ⑥ 解题技巧 → ⑦ 真题例子 → ⑧ 高频考点总结

## 专项刷题

用户输入语法点名称，系统自动生成阶梯练习：

**基础题 → 提高题 → 北京真题 → 综合题 → 易错题 → 模拟题**

每套可选：10题 / 20题 / 30题 / 50题 / 100题

题型包括：单选、填空、改错、句型转换、翻译、阅读、写作

自动附答案 + 解析。

## 自动生成

# AGENT 4 — 智能阅卷系统（Paper Analyzer）

支持上传：PDF、Word、图片、扫描件、手机拍照

## 工作流程

```
OCR识别 → 识别学生答案 → 识别标准答案 → 自动批改 → 自动评分 → 生成分析报告
```

## 分析内容

总分、各模块得分、正确率、知识点统计、错误原因、薄弱知识点、易错知识点、学习建议、预计提升空间

## 错题处理

① 正确答案 → ② 为什么错 → ③ 正确思路 → ④ 所属知识点 → ⑤ 北京考试考法 → ⑥ 相似练习 → ⑦ 举一反三训练

最终生成：《学生英语能力分析报告》

## 功能

输出：
① 原文 → ② 修改版 → ③ 高分范文 → ④ 修改原因 → ⑤ 可积累表达
